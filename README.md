# Image Processing Basics (MATLAB)

This repository provides a structured introduction to fundamental image processing techniques using MATLAB. It is designed for beginners, students, and researchers who want to understand core concepts and practical implementations in digital image processing.

---

## 📌 Overview

The project demonstrates essential image processing operations, including:

* Image reading and visualization
* Image preprocessing and enhancement
* Noise reduction and filtering
* Edge detection techniques
* Histogram analysis
* Image transformations

All implementations are written in MATLAB using built-in functions and simple scripts for clarity.

---

## 🧠 Topics Covered

* Grayscale and RGB image handling
* Image resizing and normalization
* Smoothing filters (Gaussian, Median)
* Edge detection (Sobel, Prewitt, Canny)
* Histogram equalization
* Thresholding and segmentation basics
* Morphological operations

## ▶️ How to Run

1. Open MATLAB
2. Navigate to the project folder:

   ```matlab
   cd 'path_to_project_folder'
   ```
3. Run any script, for example:

   ```matlab
   run('scripts/edge_detection.m')
   ```
